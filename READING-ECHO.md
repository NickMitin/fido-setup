# Как читать эху Nick Mitin says in FidoNet

Эта инструкция для тех, кто хочет читать эхоконференции, распространяемые через ноду **Nick Mitin says in FidoNet**.

---

## Информация о ноде

| Параметр | Значение |
|----------|----------|
| **AKA (адрес)** | 2:5023/24.4421 |
| **Система** | Nick Mitin says in FidoNet |
| **Локация** | Russia, Moscow |
| **Хост** | fido.nickmitin.ru |
| **Uplink** | wfido.ru (2:5023/24) |

---

## Способ 1: Веб-консоль (TTYd)

Самый простой способ — подключиться к веб-терминалу ноды и читать эху через консольные программы (GoldEd+, Blue Wave и др.).

### Шаги

1. Откройте в браузере:
   ```
   http://fido.nickmitin.ru:24580
   ```
   (или `http://<IP-адрес>:24580`, если DNS не настроен)

2. Введите пароль доступа (получите его у системного оператора ноды).

3. После входа вы окажетесь в консоли. Можно использовать:
   - **GoldEd+** — полноценный офлайн-редир;
   - **hpt scan** / **hpt toss** — для сканирования и разноса почты;
   - **squish** — база сообщений Husky.

4. Сообщения эхи лежат в `/var/spool/ftn/msgbased/...` (или эквивалентные пути Husky).

---

## Способ 2: Подписка через свой uplink

Если у вас есть собственная нода или точка с uplink’ом, который получает нужную эху от **wfido.ru** (2:5023/24):

1. Убедитесь, что ваш uplink распространяет нужную эху (например, **NICKMITIN.SAYS**).

2. Подпишитесь через **Areafix**:
   ```
   SUB NICKMITIN.SAYS
   ```
   (замените NICKMITIN.SAYS на имя нужной эхи)

3. Пароль Areafix — уточните у своего uplink’а.

4. После подписки эха будет приходить в ваш почтовый пакет при обмене с uplink’ом.

---

## Способ 3: Регистрация как point

Если вы хотите читать эху как **point** этой ноды:

1. Свяжитесь с сисопом ноды (Nick Mitin) для получения point-адреса (например, 2:5023/24.4421.1).

2. Получите пароли для BinkP-сессии и Areafix.

3. Настройте свой почтовик (BinkD, qico и др.) на обмен с нодой:
   - **Хост:** fido.nickmitin.ru
   - **Порт:** 24554 (BinkP)

4. Подпишитесь на эхи через Areafix (как в способе 2).

## Доступные эхи

Основная эха, используемая в этом проекте:
